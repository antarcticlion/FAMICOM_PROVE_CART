# FAMICOM_PROVE_CART
Famicom 60pin cartridge Probing Board. (For Logic analyzer / Oscilloscope / Multimeter / etc.)

## 概要

ファミコン本体とカートリッジ間に設置して信号をモニターするための基板です。
プロジェクトにはKiCad PCB 5.0用のプロジェクトファイルとガーバーが含まれます。
